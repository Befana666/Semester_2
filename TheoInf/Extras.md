>[!definition]+ SOS (**S**um **o**f **S**ubset)
>___
>The ==Sum of Subsets (SOS)== problem is a problem in Computer Science and Discret Mathematics.
>
>Given:
>- A set of positive integers
>  $S=\{w_1,\ w_2,\ w_3,\ .....,\ w_n\}$
>- A target value $M$
>  
>  Find all subsets whose sum equals $M$
>  
>  $S=\{5,10,12,13,15,18\},\ M=30$
>
>Valid subsets would be:
>- $\{12,18\}$
>- $\{5,10,15\}$
>- $\{5,12,13\}$
>
>___
>$$\sum\limits_{i = 1}^{n} {a_i}=\ M$$

>[!definition] Partition
>___
>The Partition problem is a classic problem in computer science, especially in **dynamic programming** and **NP-completeness**.
>
>Given:
>Given is a set of integers, determine whether it can be divided into two subsets such that the sum of the elements in both subsets is equal.
>
>Example:
>	Input:
>		\[1, 5, 11, 5\]
>		
>	Output:
>		True
>		
>	- Subset 1: \[1, 5, 5\] $\rightarrow$ sum = 11
>	- Subset 2: \[11\] $\rightarrow$ sum = 11
>___
